Variáveis e Entrada de Dados

Capítulo 3: Aprofundando os Conceitos

Regras para Nomes de Variáveis

Válidos ✔️

  • a1
  • velocidade
  • salario_medio

Inválidos ❌

  • 1a (não pode começar com número)
  • salario medio (não pode ter espaço)

Tipos de Variáveis

  • Inteiro (int): Números sem parte decimal. Ex: 10, -5, 1500.
  • Ponto Flutuante (float): Números com parte decimal. Ex: 1.5, -0.75. Use ponto, não vírgula!
  • Lógico (bool): Armazena apenas True (verdadeiro) ou False (falso).
  • String (str): Sequências de texto. Ex: "Olá, mundo!".

Operadores Relacionais (Comparações)

Usados para comparar valores. O resultado é sempre True ou False.

>>> a = 10
>>> b = 20

>>> a == 10    # Igual? -> True
>>> a != b      # Diferente? -> True
>>> b > a       # Maior que? -> True
>>> a <= 10    # Menor ou igual? -> True

Operadores Lógicos

Combinam expressões lógicas.

# 'and': ambos precisam ser True
>>> (10 > 5) and (3 < 4)   # True and True -> True

# 'or': pelo menos um precisa ser True
>>> (10 < 5) or (3 < 4)    # False or True -> True

# 'not': inverte o valor
>>> not (10 > 5)            # not True -> False

Operações com Strings

  • Concatenação (+): Junta duas strings. "Olá" + " " + "Mundo" resulta em "Olá Mundo".
  • Repetição (*): Repete uma string. "eco" * 3 resulta em "ecoecoeco".
  • Fatiamento ([ ]): Pega um pedaço da string. "Python"[0:2] resulta em "Py".

Entrada de Dados: `input()`

A função input() pausa o programa e espera o usuário digitar algo.

# O texto digitado é guardado na variável 'nome'
nome = input("Qual é o seu nome? ")

print("Olá, " + nome + "!")

Convertendo a Entrada de Dados

input() sempre retorna texto (string). Para usar como número, precisamos converter.

# Converte a string para um número inteiro
idade_str = input("Digite sua idade: ")
idade_int = int(idade_str)

# Converte a string para um número decimal
altura_str = input("Digite sua altura: ")
altura_float = float(altura_str)
