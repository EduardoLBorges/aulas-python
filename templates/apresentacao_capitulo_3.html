Variáveis e Entrada de Dados

Regras para Nomes de Variáveis

Permitido ✔️

  • Começar com letra ou underline (_).
  • Conter letras, números e underline.
  • Ex: velocidade, nota_final, _temp

Proibido ❌

  • Começar com número.
  • Conter espaços ou caracteres especiais (-, +, *, /).
  • Ex: 1nota, salario medio, valor-final

Tipos de Variáveis: Números

Inteiros (int)

Números completos, sem parte decimal.

idade = 30
ano = 2023

Ponto Flutuante (float)

Números que possuem parte decimal. Use o ponto (.) como separador.

altura = 1.75
preco = 19.99

Tipos de Variáveis: Lógico (bool)

Armazena apenas um de dois valores: True (Verdadeiro) ou False (Falso). É o resultado de toda comparação.

aprovado = True fim_de_jogo = False maior_de_idade = (18 >= 18) # Resulta em True

Operadores Relacionais

Comparam valores e retornam True ou False.

== # Igual a != # Diferente de > # Maior que < # Menor que >= # Maior ou igual a <= # Menor ou igual a

Operadores Lógicos: `and`, `or`, `not`

Combinam expressões booleanas.

# and: Verdadeiro SÓ SE AMBOS forem verdadeiros >>> (5 > 3) and (10 < 20) # True and True True # or: Verdadeiro SE PELO MENOS UM for verdadeiro >>> (5 < 3) or (10 < 20) # False or True True # not: Inverte o resultado >>> not (5 > 3) # not True False

Tipos de Variáveis: String (str)

Strings são sequências de caracteres (texto), delimitadas por aspas simples ('') ou duplas ("").

nome = "Maria" mensagem = 'Bem-vindo ao curso!'

Operações com Strings: Juntar e Repetir

# Concatenação (+) >>> nome = "José" >>> sobrenome = "Silva" >>> nome_completo = nome + " " + sobrenome >>> print(nome_completo) José Silva # Repetição (*) >>> print("-" * 10) ----------

Composição: Formatando Strings

Permite inserir valores de variáveis dentro de um texto modelo.

nome = "Ana" idade = 25 saldo = 102.50 # %s para string, %d para inteiro, %f para float # %.2f formata o float com 2 casas decimais print("%s tem %d anos e R$ %.2f na conta." % (nome, idade, saldo)) # Saída: Ana tem 25 anos e R$ 102.50 na conta.

Fatiamento: "Cortando" Pedaços

Acessa partes de uma string usando índices. O primeiro caractere está no índice 0.

>>> s = "PYTHON" >>> s[0] # Primeiro caractere 'P' >>> s[0:2] # Do índice 0 até o 2 (sem incluir o 2) 'PY' >>> s[2:] # Do índice 2 até o final 'THON' >>> s[-1] # Último caractere 'N'

Entrada de Dados com `input()`

A função input() pausa o programa e espera o usuário digitar algo. Ela sempre retorna uma string.

nome = input("Digite seu nome: ") print("Olá, " + nome)

Conversão de Tipos (Casting)

Como input() retorna string, precisamos converter para usar como número.

# Lê a idade como string idade_str = input("Digite sua idade: ") # Converte para inteiro idade_int = int(idade_str) # Lê o valor como string valor_str = input("Digite o preço: ") # Converte para float valor_float = float(valor_str)
