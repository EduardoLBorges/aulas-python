Capítulo 2: Instalação e Primeiros Passos

O que é o Interpretador Python?

É o programa essencial que lê os seus comandos e os "traduz" para que o computador possa executá-los.

Instalação do Python 3

Para começar, precisamos da versão 3 do Python.

  1. Acesse python.org e vá para Downloads.
  2. Baixe o instalador para o seu sistema (Windows, Mac, Linux).
  3. Siga os passos da instalação, mantendo as opções padrão.

Conhecendo o IDLE (Python Shell)

O IDLE é um ambiente interativo para testar comandos e ver o resultado na hora.

>>> print("Olá, Mundo!")
Olá, Mundo!
>>>

Criando e Executando um Programa

  1. No IDLE: File > New File.
  2. Digite seu código.
  3. Salve o arquivo com a extensão .py.
  4. Pressione F5 ou vá em Run > Run Module.

Cuidados ao Digitar

  • Maiúsculas/Minúsculas: print ≠ Print.
  • Aspas: Sempre abra e feche aspas (" " ou ' ').
  • Parênteses: Todo ( deve ter um ) correspondente.
  • Indentação: Espaços no início da linha são cruciais.

Python como Calculadora

Faça contas direto no interpretador.

>>> 2 + 3
5
>>> 10 * 5
50
>>> 2 ** 3 # Exponenciação
8
>>> 10 % 3 # Resto da divisão
1

Variáveis e Atribuição

Variáveis guardam valores. Use = para atribuir.

salario = 1500
aumento = 5

novo_salario = salario + (salario * aumento / 100)

>>> print(novo_salario)
1575.0
