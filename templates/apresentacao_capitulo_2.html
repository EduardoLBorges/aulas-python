Capítulo 2

Preparando o Ambiente em Detalhes

O Interpretador Python

É o "cérebro" que executa seus programas. Sem ele, um código Python é apenas um texto.

  • Traduz: Converte seus comandos para uma linguagem que o computador entende.
  • Executa: Realiza as ações, linha por linha.
  • Verifica: Aponta erros de sintaxe se você escrever algo errado.

Instalação: Visão Geral

O primeiro passo é ter o Python 3 instalado.

  • Fonte Oficial: Baixe sempre de python.org.
  • Windows: Requer instalação manual.
  • Linux & Mac OS X: Geralmente já vêm com uma versão, mas pode ser necessário instalar a versão 3.

IDLE: Seu Ambiente Inicial

O IDLE é um ambiente simples que vem com o Python, perfeito para começar. Ele tem duas janelas principais.

IDLE: Shell vs. Editor

Python Shell (Interativo)

É a janela com o sinal >>>. Ótima para testes rápidos. Você digita um comando, aperta Enter e vê o resultado na hora.

Editor de Arquivos

É uma janela em branco (File > New File) onde você escreve programas completos para salvar e executar várias vezes.

Usando o Shell como Calculadora

Você pode fazer contas diretamente no Shell.

>>> 10 + 5 # Soma 15 >>> 20 - 8 # Subtração 12 >>> 7 * 3 # Multiplicação 21 >>> 50 / 4 # Divisão 12.5

Mais Operações Matemáticas

# Exponenciação (2 elevado a 8) >>> 2 ** 8 256 # Resto da divisão (Módulo) >>> 10 % 3 1

Variáveis: Guardando Valores

Variáveis são "etiquetas" que você cria para armazenar dados. Use o sinal = para atribuir um valor.

# A variável 'salario' agora guarda o valor 1500 >>> salario = 1500 # A variável 'aumento' guarda 5 >>> aumento = 5 # Podemos usar as variáveis em cálculos >>> print(salario + (salario * aumento / 100)) 1575.0

Comentários: Suas Anotações

Tudo que vem depois do símbolo # em uma linha é ignorado pelo Python. Use para explicar o que seu código faz.

# Este é um comentário. Ele não será executado. # Define o salário inicial do funcionário salario = 1500 # Atribui 1500 à variável salario

Fim da Revisão
